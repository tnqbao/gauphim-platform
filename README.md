## 🚀 Hướng dẫn cài đặt

### 1. Clone toàn bộ project:
```bash
  ./init.sh
```

## 2. Cập nhật các submodules (frontend & backend):
```bash
  git submodule update --init --recursive
```

## 3. Hoặc cập nhật riêng từng phần:

### ➤ Cập nhật Frontend:

```bash
  update-frontend.sh
```

### ➤ Cập nhật Backend:

```bash
  update-backend.sh
```
